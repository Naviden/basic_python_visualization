# Python Visualization Basics

This repository provides a short tutorial on the basics of visualization in Python, created for a 2-hour lesson. The main objective is to introduce the fundamentals of Matplotlib and guide you through building several plots from scratch to practice the essential components of Matplotlib.

## Requirements

Install the required libraries:

```bash
pip install -r requirements.txt
```

This will install: numpy, pandas, matplotlib, seaborn, and bokeh.

## How to Use This Repository

1. **Begin with Matplotlib Introduction:**
   If you are new to Python and/or Matplotlib, start with the [Matplotlib Introduction](https://github.com/Naviden/basic_python_visualization/blob/master/An%20Introduction%20to%20Matplotlib.ipynb). This notebook covers the basics of the library including plot anatomy, line plots, styling, subplots, and saving figures.

2. **Study the First Case Study:**
   Once you are familiar with Matplotlib, proceed to the [first case study](https://github.com/Naviden/basic_python_visualization/blob/master/Case%20Study/Case_Study_1.ipynb). This lesson guides you step-by-step in creating histograms, scatter plots, and time-series visualizations using the Bike-Sharing dataset:
   ![Final Plot](https://github.com/Naviden/basic_python_visualization/blob/master/images/all_final.png)

3. **Practice with the Second Case Study:**
   After completing the first case study, move on to the [second case study](https://github.com/Naviden/basic_python_visualization/blob/master/Case%20Study/Case_Study_2_student.ipynb). This will help you practice creating the following plot:
   ![Final Plot 2](https://github.com/Naviden/basic_python_visualization/blob/master/images/final_2.png)

4. **Additional Practice:**
   Try the [ChickWeight Practice](https://github.com/Naviden/basic_python_visualization/blob/master/Practice/ChickWeight%20Practice.ipynb) notebook to work with box plots and grouped data analysis.

5. **Explore Other Visualization Libraries:**
   Check out the [Other Libraries](https://github.com/Naviden/basic_python_visualization/blob/master/Other%20Tools/Other%20Libraries.ipynb) notebook for an overview of Seaborn, Bokeh, Plotly, and Dash.

## Topics Covered

- **Matplotlib**: Plot anatomy (Figure, Axes, Axis), line plots, bar charts, histograms, scatter plots, pie charts, subplots, styling (colors, markers, linestyles), annotations, legends, and saving figures
- **Data handling**: Reading CSVs with pandas, rolling averages, outlier detection
- **Seaborn**: Statistical graphics, pairplots, jointplots
- **Bokeh**: Interactive visualizations with selection tools
- **Plotly & Dash**: Overview of interactive charting and dashboards

Good luck!
